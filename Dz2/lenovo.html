Lenovo IdeaPad

Описание товара

Идеальным выбором, как для работы, так и для развлечений будет ноутбук LENOVO IdeaPad S340-14API.

Тонкий и легкий корпус обеспечивает максимальную мобильность.

Характеристики товара

  • Диагональ = 14
  • Разрещение = 1920х1080
  • Процессор - AMD Ryzen 3
  • Видеокарта - AMD Radeon Vega 3
  • Оперативная память = 8Гб

Подробное описание товара

14-дюймовый дисплей с узкой рамкой обеспечивает качество отображаемой картинки. Современный процессор AMD Ryzen 3 3200U гарантирует быстрое и эффективное решение рабочих задач. Видеокарта AMD Radeon Vega 3 обладает мощными графическими возможностями. Емкий аккумулятор обеспечивает длительную эксплуатацию устройства в автономном режиме. Ноутбук LENOVO IdeaPad S340-14API обладает достаточным объемом памяти для хранения важных документов и мультимедийных файлов. Воспроизведение звука с эффектом погружения обеспечивает фирменная технология Dolby Audio. Для обеспечения конфиденциальности WEB-камера обладает специальной шторкой.
