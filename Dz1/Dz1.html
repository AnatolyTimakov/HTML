Подробное описание товара

В комплектацию входит процессор Intel Core i9 9880H с операционной системой Windows 10. Такие параметры гарантируют мгновенную загрузку любого файла и трехмерной игры, запрос пользователя имеет быстрый отклик, а ресурсоемкие задачи решаются без труда. Ноутбук MSI GE65 Raider 9SF-070RU оснащен твердотельным накопителем SSD на 1024 Гб для хранения большого количества информации. Для обмена данными есть возможность выхода в интернет через Wi-Fi. Благодаря встроенной камере и потрясающей акустической системе можно общаться по видеосвязи, проводить онлайн-конференции или играть большой командой.
